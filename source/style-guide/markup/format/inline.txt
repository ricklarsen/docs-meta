Basic Markup
------------

To format text using {+rst+}, you add matching sets of a specific
character around the text that you want to format.

.. list-table::
   :header-rows: 1
   :widths: 30 30 40

   * - Formatting
     - Method
     - Example

   * - **strongly emphasize** or **bold** text
     - Add two asterisks on either side of the text.
     -
       .. code-block:: rst
          :copyable: false

          **bold**

   * - *emphasize* or *italicize* text
     - Add one asterisk on either side of the text.
     -
       .. code-block:: rst
          :copyable: false

          *italic*

   * - ``monospace`` text
     - Add two backticks on either side of the text.
     -
       .. code-block:: rst
          :copyable: false

          ``monospace``

Inline Markup Limitations
-------------------------

Basic markup has some limitations. If you don't avoid these
limitations, the build tools generate errors.

.. list-table::
   :header-rows: 1
   :widths: 50 25 25

   * - Limitation
     - Use
     - Avoid

   * - No whitespace allowed between the emphasized word and the
       emphasis markup.
     - .. code-block:: rst
          :copyable: false

          **bold code**
     - .. code-block:: rst
          :copyable: false

          ** bold code**

   * - No more than one type of markup to any text.
     - .. code-block:: rst
          :copyable: false

          ``bold code``

     - .. code-block:: rst
          :copyable: false

          **``bold code``**

   * - No markup in the middle of text without escaping a space
       around the markup.
     - .. code-block:: rst
          :copyable: false

          this is *four* words

          thisis\ *one*\ word

     - .. code-block:: rst
          :copyable: false

          thisis*one*word
