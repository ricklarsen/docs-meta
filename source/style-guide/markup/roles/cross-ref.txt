
Cross Referencing Role Syntax
-----------------------------

.. list-table::
   :header-rows: 1
   :widths: 25 75

   * - Directive
     - Description

   * - ``:ref:``
     - Link to explicit reference elsewhere in repository or across
       repositories.

   * - ``:doc:``
     - Refers to file in the same repository.

   * - ``:rfc:``
     - Refers to Internet Request for Comments.
